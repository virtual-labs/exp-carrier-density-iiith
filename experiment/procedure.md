### Procedure
#### Task-1
In the first task, the user is provided with three plots of carrier distribution vs Energy. The fermi level of each of these plots is either near the midgap, below midgap or above midgap. The user is asked to match the plots to what type of dopants are present in each semiconductor.<br>

#### Task-2
In the next task, the user is provided with a plot of temperature vs the majority charge carriers. The graph is divided in 3 parts which represents three different states of carrier concentrations and the user is asked to label the graph with the provided text boxes.
